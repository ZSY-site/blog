    # map()和forEach()的区别

    参考文章1 (opens new window)

    参考文章2之foreach()中return (opens new window)

    • 共同点：
        1. ( Array.prototype.map/forEach ) 都是Array原型上的方法，都是循环遍历数组中每一项
        1. 参数都是传入一个匿名函数，匿名函数有3个参数，分别是当前项item, 当前项索引index， 原始数组。
    • 差异点:
        1. 返回值不同：map返回的是一个新数组，forEach返回值是undefined
        1. 循环过程中能否被中断：map无论如何都不能使其终止。forEach 只能在try...catch 中终止循环。

    # 数组相关方法

    • map((item, index, array) =>{}): 遍历数组，返回回调函数返回值所组成的新数组

    • reduce((prev, item, index, array) =>{}, initValue) : 累加，迭代。prev 为上次化简函数的return 值，cur 为当前值(从第二项开始)。 返回累加后的结果

    • filter : 过滤

    • find: 找到第一个符合回调返回值的项

    • findIndex: 找到第一个符合回调返回值的项的索引

    • every: 只要有一项为 false ，则整体为 false

    • some: 只要有一项为 true ，则整体为 true

    • forEach((item, index) => {}) : 无法 break ，可以用 try/catch 中 throw new Error 来停止

    • join : 通过指定连接符生成字符串

    • push / pop : 末尾推入和弹出，改变原数组。push返回改变后的数组长度，pop返回弹出项

    • unshift / shift : 数组头部推入 / 数组头部弹出，改变原数组。unshift返回改变后的数组长度，shift返回弹出项

    • slice(start, end) : 返回截断后的新数组，不改变原数组

    • splice(start, number, value): 返回删除元素组成的数组，value 为插入项，改变原数组

    • sort(fn) / reverse : 排序与反转，改变原数组

    • concat : 连接数组，不影响原数组，浅拷贝

    • indexOf / lastIndexOf(value, fromIndex) : 查找数组项，返回对应的下标

    • reverse: 反转数组

    # 实现数组分块

    • chunk(array, size): 将数组拆分成多个 size 长度的区块， 每个区块组成小数组, 整体组成一个二维数组
      • 比如：chunk([2, 3, 12, 1, 2, 1, 3, 2, 32, 4, 3, 2, 1, 2], 5) 返回 [ [ 2, 3, 12, 1, 2 ], [ 1, 3, 2, 32, 4 ], [ 3, 2, 1, 2 ] ]
      • 实现：
        • 第一步：先创建两个空数组，一个smallArray，用来保存小数组的元素。一个 bigArray 用来保存一个个的小数组
        • 第二步：遍历原数组
          • 如果这个小数组中的长度没有达到，那么就继续往里扔
          • 如果小数组的长度刚好达到，则将小数组扔到大数组，同时清空小数组
        • 第三步：最后还得把小数组扔到大数组

    # 手写实现slice

    Array.prototype.mySlice = function (start, end) {
    let result = []

    if (this.length === 0) {
        return result
    }

    start = start || 0
    end = end || this.length

    if (start < 0) start = 0
    if (end > this.length - 1) end = this.length

    for (let index = start; index < end; index++) {
        result.push(this[index])
    }

    return result
}

    # 手写实现数组分块

    Array.prototype.chunk = function (size = 1) {
    // 第一步：先创建如下两个空数组
    let bigArray = []
    let smallArray = []

    if (this.length === 0) {
        return bigArray
    }
    if (size < 1) {
        size = 1
    }

    // 遍历原数组
    this.forEach(item => {
        // 如果这个小数组中的长度没有达到，那么就继续往里扔
        if (smallArray.length < size) {
            smallArray.push(item)
        }

        // 如果小数组的长度刚好达到，则将小数组扔到大数组，同时清空小数组
        if (smallArray.length === size) {
            bigArray.push(smallArray)
            smallArray = []
        }
    })

    // 最后还得把小数组扔到大数组
    bigArray.push(smallArray)

    return bigArray
}

let arr = [2, 3, 12, 1, 2, 1, 3, 2, 32, 4, 3, 2, 1, 2]
let result = arr.chunk(3)
console.log(result)

    function union(arr1, ...arrays) {
    let result = [...arr1]

    if (arrays.length === 0) {
        return result
    }

    arrays.forEach(array => {
        array.forEach(item => {
            let flag = result.indexOf(item)
            if (flag === -1) {
                result.push(item)
            }
        });
    })

    return result
}

    function removeItem(array, ...args) {
    let result = []

    for (let index = 0; index < array.length; index++) {
        let flag = args.indexOf(array[index])
        if (flag !== -1) { // 遍历到的array中的当前项存在 args数组中
            result.push(array[index])
            array.splice(index, 1)
            index--
        }
    }

    return result
}

function dropLeft(array, count) {
    if (array.length === 0) {
        return []
    }

    if (count < 1) {
        count = 1
    }

    return array.filter((item, index) => index >= count)
}

    • 若 a 小于 b，在排序后的数组中 a 应该出现在 b 之前，则返回一个小于 0 的值。
